# Atlanta Health Data Exploration

## Team Project

This project analyzes 2024 mortality data from Fulton and DeKalb counties using Python.

### Files

* `explore_oasis.py` - Main analysis script
* `requirements.txt` - Required Python packages
* `data/` - Original Excel datasets
* `outputs/` - Generated graphs, CSVs, and interactive visualizations

### Install

```bash
pip install -r requirements.txt
```

### Run

```bash
python explore_oasis.py
```

All graphs and tables will automatically be saved in the `outputs` folder.

### Current Visualizations

* Total deaths by county
* Deaths by age
* Deaths by race
* Deaths by sex
* Top causes of death
* County comparison charts
* Interactive Plotly charts
* Pie charts
* Heatmaps

Feel free to modify the graphs or add additional analyses as we continue the project.
